=========
Insertion
=========
= ToC

The above renders a table of contents.
---
(document
    (insertion
        (insertion_prefix)
        (capitalized_word)
    )
    (paragraph
        (paragraph_segment)
    )
)

=========================
Insertion with Parameters
=========================
= ToC Table of Contents

The above renders a table of contents with a custom title.
---
(document
    (insertion
        (insertion_prefix)
        (capitalized_word)
        (parameters
            (parameter)
        )
    )
    (paragraph
        (paragraph_segment)
    )
)

====================================
Insertion with Multi-Word Parameters
====================================
= Insertion first | second argument | third

The above is an insertion with three arguments
---
(document
    (insertion
        (insertion_prefix)
        (capitalized_word)
        (parameters
            (parameter)
            (parameter)
            (parameter)
        )
    )
    (paragraph
        (paragraph_segment)
    )
)
